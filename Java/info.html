Les Différentes Normes :

  • L'EAN-13 est une norme de codes barres en 1 dimension composé de 13 chiffres. Si vous souhaitez générer un code barre suivant cette norme, insérer 12 chiffres.
  • Le QR Code est une norme de codes barres en 2 dimensions. Contrairement à la norme EAN-13, il est possible de générer un QR Code à partir de n'importe quelles données.
